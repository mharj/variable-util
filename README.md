# Repository for @avanio/variable-util packages

This repository contains the following packages:

- [@avanio/variable-util](./packages/variable-util/README.md) - A main library for working with env variables
- [@avanio/variable-util-node](./packages/variable-util-node/README.md) - A library for working with env variables in Node.js
- [@avanio/variable-util-vite](./packages/variable-util-vite/README.md) - A library for working with env variables in Vite
- [@avanio/variable-util-azurekv](./packages/variable-util-azurekv/README.md) - A library for working with env variables in Azure Key Vault
- [@avanio/variable-util-tachyon](./packages/variable-util-tachyon/README.md) - A library for working with env variables in Tachyon stores
