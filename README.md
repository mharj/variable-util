This repository contains the following packages:

- [@avanio/variable-util](https://mharj.github.io/variable-util/modules/_avanio_variable-util.html) - A main library for working with env variables
- [@avanio/variable-util-node](https://mharj.github.io/variable-util/modules/_avanio_variable-util-node.html) - A library for working with env variables in Node.js
- [@avanio/variable-util-vite](https://mharj.github.io/variable-util/modules/_avanio_variable-util-vite.html) - A library for working with env variables in Vite
- [@avanio/variable-util-azurekv](https://mharj.github.io/variable-util/modules/_avanio_variable-util-azurekv.html) - A library for working with env variables in Azure Key Vault
- [@avanio/variable-util-tachyon](https://mharj.github.io/variable-util/modules/_avanio_variable-util-tachyon.html) - A library for working with env variables in Tachyon stores
